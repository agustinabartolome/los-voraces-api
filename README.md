Inicializar el proyecto
En la terminar ejecutar el comando 
npm init -y

Instalar dependencias
En la terminal ejecutar los soguientes comandos: 
npm install express 
npm install pg 
npm install bcryptjs
npm install jsonwebtoken
npm install cors
npm install dotenv
npm install nodemon --save-dev
